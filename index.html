SKILLS

스킬 & 전문 분야

보안관제 운영을 중심으로 한 위협 탐지 · 분석 · 자동화 역량.

SIEM 운영 관리

위협 탐지 / 분석

  • 이상행위 패턴 분석
  • 웹 위·변조 모니터링
  • 인텔리전스 기반 탐지 룰 작성

자동화 / 개발

  • 보안 운영 자동화 스크립트
  • 웹 기반 관제 보조 서비스 개발
  • API 연동 및 데이터 파이프라인 구성
